## v1.1 - 윤슬낚시(Yoonseul Fishing) 브랜드 자산 및 랜딩 페이지 고도화

### 주요 변경 사항
- **브랜드 자산 최신화**: AI 이미지 생성 기술을 활용하여 2.5D 로우폴리 특유의 감성을 담은 공식 배너 이미지를 신규 제작하였습니다.
  - 이 배너는 `README.md` 및 랜딩 페이지의 메인 비주얼로 사용되며, 플레이 스토어 피처 그래픽 에셋으로도 활용됩니다.
- **파비콘 적용**: 앱 아이콘(`icon-512.png`)을 웹 랜딩 페이지의 favicon으로 가공 및 등록하여 웹 페이지의 아이덴티티를 강화하였습니다.
- **랜딩 페이지 UI 고도화**: 최신 배너 이미지를 랜딩 페이지 본문에 글래스모피즘 카드로 조화롭게 노출시켜 시각적 몰입감을 배가시켰습니다.

### 프로젝트 정리 (AI Studio 흔적 제거)
- **AI Studio 스캐폴딩 제거**: `.env` / `.env.example` / `metadata.json` / `assets/.aistudio/` 와 Secrets Gradle 플러그인, 미사용 Firebase 의존성을 모두 제거하여 완전 오프라인 게임으로 정리했습니다.
- **예제 코드 제거**: 스캐폴딩 기본 예제 테스트(`ExampleUnitTest`, `ExampleInstrumentedTest`, `ExampleRobolectricTest`, `GreetingScreenshotTest`)와 스크린샷 픽스처를 제거하고, 어종 카탈로그 무결성을 검증하는 실제 단위 테스트(`FishSpeciesTest`)로 대체했습니다.
- **네이밍 정리**: `Theme.MyApplication` → `Theme.HealingFishing`, 프로젝트명 `My Application` → `Yoonseul Fishing`.

### 빌드 및 배포
- **버전**: `versionName "1.1"`, `versionCode 2` — 정식 서명 첫 릴리즈.
- **릴리즈 키스토어 보안화**: public 저장소에 노출돼 있던 약한 키(`my-upload-key.jks`, 비밀번호 `android`)를 폐기하고 git 추적에서 제거했습니다. RSA 2048 / 강력 랜덤 비밀번호의 신규 업로드 키스토어를 `.keystore/`(gitignore, 로컬 전용)에 생성했고, 릴리즈 빌드는 `.keystore/keystore.properties`를 자동으로 읽어 서명합니다.
- **플레이 스토어 그래픽 규격화**: 아이콘(512×512 32-bit PNG)·피처 그래픽(1024×500 PNG)·파비콘을 스토어 규격에 맞는 포맷/치수로 재가공했습니다.
- **자산 추출 자동화**: `exportReleaseToDesktop` Gradle 태스크가 AAB·통합 릴리즈 노트와 함께 아이콘, 피처 그래픽, 다국어 설명서를 `Build/store-assets` 하위로 일괄 추출합니다.
