## v1.1 - 윤슬낚시(Yoonseul Fishing) 브랜드 자산 및 랜딩 페이지 고도화

### 주요 변경 사항
- **브랜드 자산 최신화**: AI 이미지 생성 기술을 활용하여 2.5D 로우폴리 특유의 감성을 담은 공식 배너 이미지를 신규 제작하였습니다.
  - 이 배너는 `README.md` 및 랜딩 페이지의 메인 비주얼로 사용되며, 플레이 스토어 피처 그래픽 에셋으로도 활용됩니다.
- **파비콘 적용**: 앱 아이콘(`icon-512.png`)을 웹 랜딩 페이지의 favicon으로 가공 및 등록하여 웹 페이지의 아이덴티티를 강화하였습니다.
- **랜딩 페이지 UI 고도화**: 최신 배너 이미지를 랜딩 페이지 본문에 글래스모피즘 카드로 조화롭게 노출시켜 시각적 몰입감을 배가시켰습니다.

### 빌드 및 배포
- **버전 범프**: 앱 빌드 버전을 `versionName "1.1"`, `versionCode 2`로 상향 조정했습니다.
- **서명 빌드 프로세스 개선**: 로컬 환경 빌드 시 `KEYSTORE_PATH` 등의 환경 변수가 지정되지 않았더라도 빌드가 원활히 진행될 수 있도록 서명 비밀번호 기본값을 `"android"`로 설정하였으며, 업로드용 로컬 키스토어(`my-upload-key.jks`)를 최초 생성 및 커밋하여 릴리즈 빌드의 안정성을 확보했습니다.
- **자산 추출 자동화**: `exportReleaseToDesktop` Gradle 태스크를 개선하여 빌드 산출물(AAB) 및 통합 릴리즈 노트 외에도 고해상도 아이콘, 피처 그래픽, 다국어 텍스트 설명서 등 플레이 스토어 리스팅에 필요한 모든 자산을 `Build/store-assets` 디렉토리 하위로 일괄 추출/정리하도록 고도화하였습니다.

### 검증
- 로컬 `test`, `bundleRelease`, `exportReleaseToDesktop` 실행 성공 확인.
- 바탕화면 `Build` 디렉토리 내 `store-assets` 생성 및 구성요소 확인 완료.
